## step 4

fastとslow,フロイドのアルゴリズムを使った解法でとき直した。

サイクルがあったら本当に見つかるかどうかはgeminiに相談して、証明をした。

---

サイクルがある場合、
サイクルに入るまでの長さをL、サイクルの長さをCとする。

1秒ごに1進むS(slow pointer)、2進むF(fast pointer)について考える。

n秒後のそれぞれの位置は(このとき順分に時間が進んでおり、n > L、SもFもループに入っているとする)、

S: n - L
F: 2n - L

このとき、ループの開始位置を0とすると、

S: n - L (mod C)
F: 2n - L (mod C)

がそれぞれループからの位置となる。

よって、以下の合同式が成り立つと仮定して、SとFが出会うことを考える。

n - L ≡ 2n - L (mod C)

変形していくと、

n ≡ 2n (mod C)  両辺にLを足す。

0 ≡ n (mod C)   両辺からnを引く。

これより、nをCで割ると、余りが0になることがわかる。

よって、nがCの定数倍であるとき、SとFは出会う。
