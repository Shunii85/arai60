# step1

## 解法1
2人であるカードゲームをしている。それぞれに数字の書かれたカードのデッキが与えられ、同じ数字でペアを作る。
たデッキには同じ数字が複数枚紛れている可能性があるが、1ペアのみ作れることとする。

1. 数字の小さい順(昇順)にそれぞれ並び替える。
2. せーのでカードを出す。
  - 数字が同じなら、ペアを作る。
  - 数字が違うなら、数字の小さいのを出した人は場のカードを捨てて、新しいカードを出す準備する。
    もう一方の人は、そのまま。
3. どちらかのカードがなくなってしまえばおわり。

制約:

```
1 <= nums1.length, nums2.length <= 1000
0 <= nums1[i], nums2[i] <= 1000
```

最悪のおおよその実行時間

- ソート:  10*10^3ステップ程度
- ペアづくり: 10^3ステップ程度

から、11*10^3の計算ステップ程度で済むので、
11*10^-6 ~ 11*10^-5
11us ~ 0.1ms

### 実装
引数をコピーで受け取ることにした。ソートするため、インプットを壊したくないという目的で。

vectorは末尾からしか削除ができないので、降順に並べてからbackから取り出すことにした。
